Go File {{.option.Version}} Help documentation

basic skills

  • Upload files: Click the upload button in the upper right corner to upload. It supports multiple file uploads and drag-and-drop upload. When you upload on the "File" page, it will be uploaded to the currently displayed path; when you upload on other pages, it will be uploaded to the default path.
  • Download the file: Click the download icon.
  • Search files: In the search bar in the upper right corner of the navigation bar, press Enter to search. Keywords can be file name, upload date, uploader and description information. Notice that the search bar automatically gets focus.
  • Delete files: Click the delete icon.

Advanced functions

  • Built-in picture bed: supports pasting and uploading multiple files, drag and drop uploading.
  • Video playback: The root directory of the displayed video file is used when the program starts --video The path specified by the parameter currently supports three formats（MP4，WebM，Ogg）。
